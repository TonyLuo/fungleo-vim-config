# FungLeo Vim Config
## Install

```shell
# bak your old config
mv ~/.vim ~/.vim.bak
mv ~/.vimrc ~/.vimrc.bak

# clone my config
git clone https:#github.com/fengcms/fungleo-vim-config

# copy my config to your home directory
cd fungleo-vim-config
cp -r .vim .vimrc ~

# open vim & ignore warning
vim 

# install plugins
:PlugInstall
```
it's all right
